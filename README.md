# NEXA — IPTV Player 📺

เครื่องเล่นทีวีออนไลน์ (IPTV) ดีไซน์โมเดิร์น ทำงานในเบราว์เซอร์ ไฟล์เดียวจบ ไม่ต้องติดตั้งอะไรเพิ่ม

![status](https://img.shields.io/badge/status-ready-7c5cff) ![type](https://img.shields.io/badge/single--file-HTML-22d3ee)

## ✨ ฟีเจอร์

- ▶️ เล่นสตรีมสด **HLS / m3u8** ได้จริง (ผ่าน hls.js)
- 📂 โหลดรายการช่องจากไฟล์ **`.m3u`** (มาตรฐาน IPTV) — ดึงชื่อช่อง โลโก้ และหมวดหมู่อัตโนมัติ
- 🔗 เพิ่มช่องจาก **URL** ทีละช่องได้
- ✅ **ตรวจสอบทุกช่อง** ในคลิกเดียว — แยกสถานะ ใช้ได้ / เสีย / เช็คไม่ได้
- 🗑️ ลบทีละช่อง · ลบเฉพาะช่องที่เสีย · ล้างทั้งหมด
- 💾 **บันทึกถาวร** — ปิดแล้วเปิดใหม่ ช่องยังอยู่ครบ (ใช้ localStorage)
- ⭐ **ช่องโปรด** — กดดาวเก็บไว้ แยกแท็บดูง่าย
- 🔍 ค้นหาช่อง · 📱 รองรับมือถือ

## 🚀 วิธีใช้งาน

### แบบที่ 1 — เปิดในเครื่องตัวเอง
ดาวน์โหลด `index.html` แล้วดับเบิลคลิกเปิดด้วยเบราว์เซอร์ (Chrome / Edge / Firefox)

### แบบที่ 2 — เปิดผ่านเว็บด้วย GitHub Pages
ดูวิธีตั้งค่าด้านล่าง แล้วเข้าผ่านลิงก์ `https://<ชื่อผู้ใช้>.github.io/<ชื่อ repo>/`

## 📋 วิธีเพิ่มช่อง

1. กดปุ่ม **"โหลด Playlist (.m3u)"** แล้วเลือกไฟล์รายการช่องของคุณ
2. หรือกด **"เพิ่มจาก URL"** วางลิงก์ `.m3u8` ของช่องสด
3. คลิกช่องในรายการด้านขวาเพื่อเล่น

> มีไฟล์ตัวอย่าง `sample-playlist.m3u` ให้ลองในโปรเจกต์นี้

## ⚙️ ตั้งค่า GitHub Pages

1. อัปโหลดไฟล์ทั้งหมดขึ้น repo ของคุณ
2. ไปที่ **Settings → Pages**
3. ที่ **Source** เลือก `Deploy from a branch`
4. เลือก branch `main` โฟลเดอร์ `/ (root)` แล้วกด **Save**
5. รอสักครู่ เว็บจะพร้อมที่ `https://<ชื่อผู้ใช้>.github.io/<ชื่อ repo>/`

## 📝 หมายเหตุ

- ระบบจำช่องผูกกับเบราว์เซอร์ที่เปิด — ย้ายเครื่องหรือล้างข้อมูลเบราว์เซอร์ ช่องจะไม่ตามไป
- การตรวจสอบช่องในเบราว์เซอร์มีข้อจำกัดเรื่อง **CORS** บางช่องที่เล่นได้จริงอาจขึ้นสถานะ "เช็คไม่ได้" (เหลือง) — ไม่ได้แปลว่าเสีย
- โปรเจกต์นี้เป็นเพียงเครื่องเล่น ไม่มีช่องหรือเนื้อหาใดมาให้ ผู้ใช้ต้องเตรียม playlist เอง
